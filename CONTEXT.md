# paper-muse

论文构思助手：给定主题，盲区扫描出「非显而易见」的切入点、跨学科理论与方法（构思幕），可再圆桌深挖或对抗审查。核心不变式：**发现力相对于这个研究者判定**，不是相对于模型语料。

## Language

**研究者画像**：
研究者「是谁」的稳定描述——领域、已有立场、熟悉的理论。跨所有扫描复用、全局一份。**不含困惑**。
_Avoid_: 画像 blob（旧义含困惑）、按论文存的 profile

**困惑**：
研究者本次「想不通什么」。与主题并列的一次性输入，喂给扫描但不进研究者画像、不做全局留存。
_Avoid_: 把困惑当画像字段

**参照系**：
研究者画像 ＋ 本次困惑，合成为盲区扫描判定「非显而易见 / 离群 / 新颖」的基准。

**主题**：
本次扫描针对的研究题目或研究问题。
