# 中文法学「有趣性」判据：从本土编辑判据重建

日期：2026-07-13　状态：**研究资产（供 #93 / #94 直接消费）**

## 0. 结论与边界

中文法学并不存在一套经实证验证、可以跨共同体通用的「有趣性」量表。本轮公开来源核验只能确认三件事：

1. 《中国法学》的官方匿名评审标准明确列出「选题价值、论文创新、论证方法、论证深度、文字规范和潜在引证率」，并要求给出论文在本学科的定位；其采用程序同时说明，最终取舍还会综合刊物定位、宗旨、特色和学科等因素。因此，这份标准是**守门程序的直接证据**，不是录用公式。
2. 《中国法学》副编审任彦以编辑审稿、选稿经验提出，选题价值首先看「有无问题意识」与「有无创新」；创新须以了解、传承已有研究与共识为前提，再超越已有研究，不能脱离基础而自说自话。这是**作者署名的编辑观点**，不是《中国法学》的正式量表。
3. 本土法学方法论材料确实支持两类不同的知识作业：教义学面向实在法的解释、建构、体系化与适用；社科法学/实证研究面向实践中的法律现象、经验材料、假说检验与因果解释。但本轮没有找到中文法学期刊编辑调查或实验，证明「体系不融贯/学说重构」或「经验前提反转」会因其自身而被感知为有趣。

据此，本资产把共同的本土底座定为：**真问题 + 相对学术圈的创新 + 论证方法/深度 + 学科定位**；再按共同体把「张力」换形。它只用于构思阶段识别**值得复审的张力/贡献线索**，不得预测录用、被引或论文质量。

### 证据层级

- **【A｜期刊官方直接】**：期刊官网公布的评审标准、办刊宗旨或编辑部规则；能证明期刊公开采用的口径，不能证明因果效果。
- **【B｜编辑署名观点】**：编辑以本人名义总结的审稿/写作经验；能证明一位本土守门人的明确判断，不能自动上升为期刊统一规则。
- **【C｜本土学术直接】**：中文法学原刊论文对某研究共同体之任务、对象与方法的论证；能证明该张力形态符合共同体的知识作业，不能直接证明「更有趣」。
- **【D｜迁移理论】**：有趣性/理论贡献的一般理论或域外守门人研究；仅作结构防呆，不能替代本土判据。
- **【证据缺口·经验判断】**：未找到本土直接证据；这是供 prompt / rubric 试用的可撤销判断，必须经真实使用或本土守门人研究校准。

### 明确排除

- **Goyanes 五维不进入本判据。** 它来自传播学编委调查，不能越过共同体边界移植到中文法学。
- **Oppenheimer 清晰度不进入本判据。** 处理流畅性所涉及的是可信度/作者能力感知，不是本资产所判的有趣性。
- 《中国法学》官方指标中的「潜在引证率」**不进入 #93/#94**：本项目不得把被引当作有趣性代理，更不得据此预测被引。
- 「文字规范」仍可由既有写作/格式检查处理，但不进入张力维度。
- 张力不能替代可辩护性。任何「高张力但最强反驳成立」的卡，都不得取得强张力或高贡献判断。

## 教义学

### 张力形态定义

**教义学张力**：在现行法源、既有概念/原则、通说或稳定裁判理由所组成的规范体系内部，指出一个可定位的**规范冲突、评价矛盾、违反体系计划的不完整状态，或旧学说无法一致处理的案型**；继而在可说明的法源与方法约束下，提出能够恢复或重画体系关联的概念、规则、例外或学说方案。

这里的张力不是为了反转而反转，也不是泛称「存在争议」。合格表述须同时回答：

- **哪里不融贯**：哪两项以上规范、原则、概念、通说或裁判后果不能同时维持？
- **为何是教义问题**：冲突如何影响解释、涵摄、请求权基础、构成要件、证明责任或法律后果？
- **如何重构**：新方案如何受法源与既有体系约束，又比旧方案更一致地处理目标案型？

**证据层级**：作为法教义学任务与方法，属【C｜本土学术直接】；作为「被中文法学读者感知为有趣」的充分条件，属【证据缺口·经验判断】。本土官方/编辑材料只直接支持问题意识、创新、方法、深度和学科定位，没有对教义学张力做守门人调查。

### 教义学｜#93 质量闸判据

以下四项是 #93 可直接采用的**全取式门槛**。`D1–D3` 任一缺失，或 `D4` 不通过，均标「弱张力」；不得因辞藻强烈而补分。该全取式阈值本身属【证据缺口·经验判断】。

1. **D1｜体系诊断可定位**：卡必须点名至少两项发生冲突/断裂的规范材料、原则、概念、通说、裁判理由或法律后果，并说明其不能同时维持的具体案型；仅说「有争议」「有漏洞」「值得研究」不命中。
   - 证据层级：【B｜编辑署名观点】问题意识 + 【C｜本土学术直接】解释/建构/体系化；「至少两项」是【证据缺口·经验判断】的可操作化。
   - 共同体特定有趣性映射：**【证据缺口·经验判断】**。
2. **D2｜重构相对既有研究成立**：卡必须说明拟修正的既有学说/处理方案，并提出一个受法源和方法约束的新概念、规则、例外或关系结构；只换术语、只罗列域外法、或完全抛开既有共识自创体系不命中。
   - 证据层级：【A｜期刊官方直接】论文创新/学科定位 + 【B｜编辑署名观点】基于传承而超越 + 【C｜本土学术直接】教义学可推导新规范/概念。
   - 共同体特定有趣性映射：**【证据缺口·经验判断】**。
3. **D3｜适用差异可推出**：卡必须指出重构前后至少一个可比较的法律适用差异，例如请求权基础、构成要件、证明责任、裁判理由或法律后果发生何种变化；只宣称「完善理论体系」不命中。
   - 证据层级：【A｜期刊官方直接】选题价值/论证深度 + 【C｜本土学术直接】教义体系服务法律适用；把它设为硬门槛属【证据缺口·经验判断】。
   - 共同体特定有趣性映射：**【证据缺口·经验判断】**。
4. **D4｜体系内 steelman 存活**：卡必须给出最强体系内反驳，并说明方案为何不违反更高位法源、基本原则、法安定性或相邻制度；若反驳成立而无回应，即使 D1–D3 很强也不得判为强张力。
   - 证据层级：【A｜期刊官方直接】论证方法/论证深度 + 【B｜编辑署名观点】严谨评价既有研究、不得为突出己见而偏激驳斥；具体 steelman 形式属【D｜迁移理论】。
   - 共同体特定有趣性映射：**【证据缺口·经验判断】**。

> **#93 prompt-ready 文本（教义学）**
> 先判断该卡是否精确定位了现行规范、原则、概念、通说或裁判后果之间的不融贯/漏洞；再判断它是否在法源与既有学说约束内提出了真正改变法律适用的重构。只有「体系诊断可定位 + 重构相对既有研究成立 + 适用差异可推出 + 最强体系内反驳后仍存活」四项同时满足，才可标强张力；否则标弱张力。

### 教义学｜#94 rubric 措辞

> **有趣性/贡献（教义学）**：中心主张是否揭示了可复述的规范冲突、教义漏洞或既有学说无法一致处理的案型，并在法源与体系约束内提出能改变具体法律适用的重构？请写明「旧体系如何失配—新方案如何恢复/重画关联—哪项适用结论因此改变」，再用最强体系内反驳复审；反驳成立而无回应时，不得判为高贡献。

**证据层级**：前半句的共同体任务属【C｜本土学术直接】；问题意识、创新与论证约束属【A｜期刊官方直接】+【B｜编辑署名观点】；将其组合成一行 rubric 属【证据缺口·经验判断】。

### 教义学｜中文法学示例

**示例卡（研究方向，不是事实结论）**：身心灵/「显化致富」付费课程纠纷通常被并列放进虚假宣传、消费欺诈与合同目的不能实现三个框架。教义学张力不写成泛泛的「应加强规制」，而写成：

> **旧体系失配**：同一项「显化能致富」陈述，在广告/反不正当竞争规范中可能先问其是否为可验证事实，在消费欺诈中还要处理经营者故意与消费者信赖，在合同解除中则转向给付内容与合同目的；若三个框架对事实/信念、证明责任及救济后果的分类彼此脱节，同类案型可能得到难以融贯的处理。
> **重构方向**：以「可验证的具体功效承诺—不可验证的信念表达—未履行的服务给付」为第一层，再按各请求权的特殊要件分流；检验该两层结构能否同时说明责任成立与救济差异。
> **适用差异**：同一句营销表述不会因被统称为「玄学」而一律落入或排除某一请求权，法院须分别说明可验证性、信赖/因果关系、给付履行与救济基础。
> **最强反驳**：三套制度的规范目的本来不同，不同后果未必是不融贯；研究必须证明其所指的是无理由冲突，而非合理分工。

命中：D1（跨制度分类/后果冲突）+ D2（两层分流重构）+ D3（证明与请求权适用发生差异）+ D4（回应「规范目的不同」）。示例素材来自 [`examples/seeds/manifestation-x-law.md`](../../examples/seeds/manifestation-x-law.md)，不预断任何真实案件的裁判结果。

## 社科法学

### 张力形态定义

**社科法学张力**：先点名某个法律共同体用来解释制度运行、行为者选择或治理效果的**经验默认前提 P**，再用可核验资料显示 P 不成立、仅在特定条件下成立，或实际机制主要由另一变量/过程解释；最后说明这一修正如何改变一个法学解释、规范评价或制度设计命题。

合格表述不是「发现一个冷知识」，而须同时回答：

- **谁默认什么**：哪类学者、实务者或制度话语把哪个经验命题当作理所当然？
- **什么证据构成反转**：样本、观察、访谈、文本/裁判数据或比较材料如何否定、限缩或替换 P？
- **法学上为何重要**：修正 P 后，哪个解释、规范评价或制度设计应随之调整？

**证据层级**：经验对象、假说检验与从经验提炼理论属【C｜本土学术直接】；「反转经验默认前提」作为有趣性张力的形式来自【D｜迁移理论】；其在中文社科法学守门人中的排序与阈值均属【证据缺口·经验判断】。

### 社科法学｜#93 质量闸判据

以下四项是 #93 可直接采用的**全取式门槛**。`S1–S3` 任一缺失，或 `S4` 不通过，均标「弱张力」；「结论反常识」本身不加分。该全取式阈值本身属【证据缺口·经验判断】。

1. **S1｜默认经验前提可归属**：卡必须把 P 写成可被资料支持或反驳的经验命题，并说明它属于哪类学术/制度话语；只说「传统观点认为」「人们通常以为」而无对象和命题不命中。
   - 证据层级：【B｜编辑署名观点】问题须从现实/立法/司法/理论中提炼 + 【C｜本土学术直接】经验事实/理论问题；要求归属到共同体属【D｜迁移理论】。
   - 共同体特定有趣性映射：**【证据缺口·经验判断】**。
2. **S2｜反转有资料路径与边界**：卡必须给出能检验 P 的资料、样本/案例选择与比较方式，并明确预期结论是「否定、限缩条件或替代机制」中的哪一种；只有轶事、单个惊奇案例或先写结论再找材料不命中。
   - 证据层级：【A｜期刊官方直接】论证方法/论证深度 + 【C｜本土学术直接】观察收集资料、提出/检验假说、个案普遍性限制。
   - 共同体特定有趣性映射：**【证据缺口·经验判断】**。
3. **S3｜法学推论可导出**：卡必须指出经验修正将改变哪个法律解释、规范评价或制度设计，并给出从证据到该推论的中间机制；只报告相关性或现象频率、不回到法学问题不命中。
   - 证据层级：【A｜期刊官方直接】选题价值/学科定位 + 【B｜编辑署名观点】真问题/创新 + 【C｜本土学术直接】从经验到理论、制度效果与改革措施。
   - 共同体特定有趣性映射：**【证据缺口·经验判断】**。
4. **S4｜替代解释与证据 steelman 存活**：卡必须列出最强替代解释、样本偏差/测量误差或反例，并说明现有证据最多支持到描述、相关、机制线索还是因果；越过证据层级，或反驳成立而无回应，不得判为强张力。
   - 证据层级：【A｜期刊官方直接】论证方法/深度 + 【B｜编辑署名观点】严谨客观评价 + 【C｜本土学术直接】实证/社科方法自身局限；具体 steelman 形式属【D｜迁移理论】。
   - 共同体特定有趣性映射：**【证据缺口·经验判断】**。

> **#93 prompt-ready 文本（社科法学）**
> 先判断该卡是否点名了某共同体可检验的经验默认前提，再判断资料能否否定、限缩或以替代机制修正该前提，并能否由此导出明确的法律解释、规范评价或制度设计变化。只有「默认前提可归属 + 反转有资料路径与边界 + 法学推论可导出 + 最强替代解释/证据反驳后仍存活」四项同时满足，才可标强张力；否则标弱张力。

### 社科法学｜#94 rubric 措辞

> **有趣性/贡献（社科法学）**：中心主张是否点名并以可核验资料否定、限缩或替换了某共同体的经验默认前提，且该修正足以改变一个法律解释、规范评价或制度设计？请写明「默认前提 P—证据与边界—修正后的机制 Q—法律推论」，再用最强替代解释、样本/测量局限与反例复审；证据只到描述或相关时，不得写成因果，也不得判为高贡献。

**证据层级**：经验检验、证据边界与法学回扣属【C｜本土学术直接】；问题意识、创新与论证约束属【A｜期刊官方直接】+【B｜编辑署名观点】；「默认前提 P → 修正机制 Q」作为有趣性 rubric 属【D｜迁移理论】+【证据缺口·经验判断】。

### 社科法学｜中文法学示例

**示例卡（待检验假说，不是事实结论）**：沿用身心灵/「显化致富」付费课程纠纷，但把张力放在经验前提而非规范体系：

> **默认前提 P**：相关法律讨论可能默认，案件成败主要取决于法院是否把「显化能致富」认定为可验证的事实陈述，而不是信念表达。
> **资料路径**：建立公开裁判文书与监管决定的可复核样本，编码当事人主张、宣传语类型、实际给付、证据形态、请求权基础、法院未支持的理由与救济结果；另以负例检索和双人复核控制只收集「玄学」关键词所造成的选择偏差。
> **可能的反转 Q**：若多数裁判实际绕开事实/信念定性，而主要因合同相对性、宣传主体、信赖/因果证明或服务已经履行而处理案件，则 P 只解释少数案型；真正机制可能是举证与请求权结构。
> **法律推论**：规制讨论应从抽象地扩张「虚假」定义，转向区分宣传可验证性、证明责任与不同请求权的证据门槛。
> **最强替代解释**：公开文书可能遗漏调解/撤诉案件，关键词检索也可能系统漏掉未使用「显化」标签的纠纷；在这些限制排除前，只能称为样本内模式，不能声称总体因果。

命中：S1（可归属、可检验的 P）+ S2（文书/监管样本与反转条件）+ S3（指向证明责任和请求权结构）+ S4（公开文书与检索偏差）。示例仅把 [`examples/seeds/manifestation-x-law.md`](../../examples/seeds/manifestation-x-law.md) 的「裁判文书量化」方向改写为可证伪设计，不声称已经取得数据或验证 Q。

## 2. 尚待验证的证据缺口

1. **无本土守门人分组调查**：没有找到对中文法学编辑/外审专家按教义学、社科法学分组的判据调查；两套阈值、相对重要性与失败样例均待本土复制研究。
2. **无因果证据**：没有实验或准实验显示，体系不融贯/经验前提反转会因其自身提高中文法学读者的感知有趣性。
3. **共同体边界会重叠**：许多论文同时包含规范解释与经验研究。当前「据中心主张的主要证成方式归类」是【证据缺口·经验判断】，真实使用时应允许双标签，但不得合并为一个总分。
4. **自动质量闸只能识别文本契约**：`D1–D4` / `S1–S4` 可检查卡是否把张力说完整，不能自动验证法源穷尽、资料真实性、研究设计质量或学说的新颖性。任何真实判断仍需检索和专家复审。
5. **示例尚未做真实样本研究**：两枚示例都是 prompt-ready 的可证伪研究方向，不是对裁判实践或现行法的已证实结论。
